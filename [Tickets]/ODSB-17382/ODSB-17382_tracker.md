### Schema Gotchas Discovered
- `req.device.country` does NOT exist in `prod_stream_view.imp` — derive country from `campaign_id` in Python
- `cv` table fields are nested under `cv.pb.*` (not top-level): `cv.pb.moloco.campaign_id`, `cv.pb.device.model`, `cv.pb.event.name`
- `pb` table uses top-level `moloco.campaign_id`, `device.model`, `event_name` — different from `cv`
- CPA must use device-level spend from imp table (`imp.cost.analysis...`), not prorated from `fact_dsp_all`
