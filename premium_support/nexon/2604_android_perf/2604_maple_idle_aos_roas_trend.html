SQL Queries — Replication Reference

Both queries run against moloco-ae-view.athena.fact_dsp_core on project moloco-ods. Click a header to expand, then use Copy to grab the SQL.

Query 1 — Overall Weekly D3 & D7 ROAS

Returns 1 row per week · aggregates all geos · 18 rows returned
▶ 
SELECT
  DATE_TRUNC(date_utc, WEEK(MONDAY))                                         AS week_start,
  SUM(gross_spend_usd)                                                         AS spend_usd,
  SUM(installs)                                                                AS installs,
  SAFE_DIVIDE(SUM(revenue_d3),         SUM(gross_spend_usd))                  AS d3_roas,
  SAFE_DIVIDE(SUM(revenue_d7),         SUM(gross_spend_usd))                  AS d7_roas,
  SAFE_DIVIDE(SUM(capped_revenue_d3),  SUM(gross_spend_usd))                  AS d3_croas,
  SAFE_DIVIDE(SUM(capped_revenue_d7),  SUM(gross_spend_usd))                  AS d7_croas
FROM `moloco-ae-view.athena.fact_dsp_core`
WHERE
  product.app_market_bundle = 'com.nexon.ma'
  AND campaign.os           = 'ANDROID'
  AND date_utc              >= DATE('2026-01-01')
GROUP BY 1
ORDER BY 1

Query 2 — Weekly D3 & D7 ROAS by Geo

Returns 1 row per week × geo · focus geos + Other bucket · 144 rows returned
▶ 
WITH base AS (
  SELECT
    DATE_TRUNC(date_utc, WEEK(MONDAY)) AS week_start,
    CASE
      WHEN campaign.country IN ('KOR', 'TWN', 'USA', 'THA', 'CAN', 'SGP', 'MYS') THEN campaign.country
      ELSE 'Other'
    END AS geo,
    SUM(revenue_d3)         AS rev_d3,
    SUM(revenue_d7)         AS rev_d7,
    SUM(capped_revenue_d3)  AS cap_rev_d3,
    SUM(capped_revenue_d7)  AS cap_rev_d7,
    SUM(gross_spend_usd)    AS spend_usd
  FROM `moloco-ae-view.athena.fact_dsp_core`
  WHERE
    product.app_market_bundle = 'com.nexon.ma'
    AND campaign.os           = 'ANDROID'
    AND date_utc              >= DATE('2026-01-01')
  GROUP BY 1, 2
)
SELECT
  week_start, geo,
  SAFE_DIVIDE(rev_d3,     spend_usd) AS d3_roas,
  SAFE_DIVIDE(rev_d7,     spend_usd) AS d7_roas,
  SAFE_DIVIDE(cap_rev_d3, spend_usd) AS d3_croas,
  SAFE_DIVIDE(cap_rev_d7, spend_usd) AS d7_croas,
  spend_usd
FROM base
ORDER BY 1, 2
